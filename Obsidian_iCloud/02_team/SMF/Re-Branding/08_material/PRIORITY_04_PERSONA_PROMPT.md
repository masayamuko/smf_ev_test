#### 情報収集行動
```
【使用デバイス】
スマートフォン（iPhone）> ノートPC

【よく見るSNS・サイト】
Instagram（留学アカウントフォロー）
YouTube（体験談動画）

友人・先輩の体験談
```

#### 留学検討プロセス
```
【認知段階】

「私にもできるかな？」と憧れ

【情報収集段階】
Google検索で基本情報収集
「初めて留学」「英語できない 留学」
複数のエージェント比較

【検討段階】
家族・友人に相談
費用・安全性を重視
無料カウンセリング参加

【決定段階】
信頼できるカウンセラーとの出会い
具体的なプラン・サポート内容確認
家族の理解・安心材料確保
```

### 3. ペインポイント・ニーズ

#### 主要なペインポイント
```
🤔 知識不足による不安
「何から始めればいいか分からない」
「パスポートの取り方も分からない」
「留学の種類も違いが分からない」

😰 英語力への不安
「英語ができないのに海外で生活できる？」
「現地で困った時にどうしよう」
「授業についていけるか心配」

💰 費用面での心配
「留学費用はどのくらいかかる？」
「隠れた費用はないか心配」
「親に負担をかけすぎないか」

👥 人間関係への不安
「現地で友達ができるか心配」
「ホストファミリーと合うか不安」
「日本人が少なすぎても困る」

🛡️ 安全面への懸念
「治安は大丈夫？」
「病気や怪我をした時の対応」
「緊急時の連絡手段」
```

#### 求めるサポート内容
```
✅ 段階的な説明・指導
基礎知識から丁寧に教えてほしい
一度に多くの情報は処理できない

✅ 心理的サポート
不安を受け止めて共感してほしい
「大丈夫」と安心させてほしい
具体的な成功事例を聞きたい

✅ 継続的なフォロー
準備期間中も定期的に連絡
現地でも困った時にすぐ相談
帰国後も相談に乗ってほしい

✅ 家族への説明サポート
親が安心できる説明・資料
緊急時の連絡体制明示
実績・安全性の具体的説明
```
