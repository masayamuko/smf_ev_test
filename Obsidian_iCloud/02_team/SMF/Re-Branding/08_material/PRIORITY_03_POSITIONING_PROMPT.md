## 📊 詳細情報（マップに含める内容）

### 競合分析データ

#### 大手留学エージェント（例：EF Education）
```
【ポジション】総合型・高品質
【強み】
- 豊富な実績・選択肢
- グローバルネットワーク
- 高品質なサービス

【弱み】
- 初心者への特化なし
- 大規模故の画一化
- 高価格帯

【ターゲット】全年齢層・多様なニーズ
```

#### 格安エージェント（例：スマ留等）
```
【ポジション】総合型・コスト重視
【強み】
- 低価格
- シンプルなパッケージ
- 若年層向けマーケティング

【弱み】
- サポートの薄さ
- 初心者への配慮不足
- 質のばらつき

【ターゲット】コスト重視・ある程度自立可能
```

#### 特化型小規模（例：地域密着型）
```
【ポジション】特化型・基本サポート
【強み】
- 地域密着
- 個人的関係
- 柔軟な対応

【弱み】
- リソース限界
- 専門性の深さ不足
- スケール困難

【ターゲット】地域限定・紹介中心
```

1. **初心者専門化**
   - パスポート取得からサポート
   - 「何も分からない」状態から対応
   - 初心者特有の不安を100%理解

2. **カウンセラーの人間力**
   - 親身・フレンドリー・誠実

3. **手厚いサポート体制**
   - 他社では真似できない面倒見
   - 留学前・中・後の継続支援
   - アットホームなコミュニティ
