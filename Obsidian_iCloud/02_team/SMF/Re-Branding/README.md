# スマフラ留学リブランディング・ツールキット

## 🚀 プロジェクト概要

### 目的
既存のスマフラ留学を「**初海外・初留学専門**」エージェントとして完全リブランディングし、初心者層での圧倒的なポジションを確立する。

### 期間
2024年7月26日 - 8月31日（約5週間）

### チーム体制
- **マサヤ（IT担当）**: システム開発、ウェブサイト、AI活用
- **ヤミー（留学カウンセラー）**: ブランド戦略、メッセージング、プログラム設計  
- **ユメナ（留学カウンセラー）**: カスタマーエクスペリエンス、コンテンツ制作

### 成功指標
- 9月末までに初心者からの問い合わせ月20件達成
- 顧客満足度4.5/5.0以上維持
- ウェブサイト滞在時間2倍向上（3分以上）

## 🚨 緊急時・トラブル対応

### 問題発生時の手順
1. **即座に記録**: 該当ファイルに状況を記録
2. **チーム共有**: Slackで即座に報告
3. **対応策検討**: `BEST_PRACTICES.md`の該当箇所を参照
4. **意思決定**: `DECISION_LOG.md`に判断プロセスを記録

### エスカレーション基準
- **レベル1**: 個人で判断可能 → 事後報告
- **レベル2**: 2名以上で協議 → 即座に相談
- **レベル3**: 全員合意必要 → 緊急ミーティング
