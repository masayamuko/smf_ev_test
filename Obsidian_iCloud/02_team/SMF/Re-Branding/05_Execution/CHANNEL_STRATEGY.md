# チャネル戦略

## 🎯 チャネル戦略の目的

### 主要目標
1. **認知拡大**: 初心者層への効率的なリーチ
2. **信頼構築**: 多面的な情報発信による信頼性向上
3. **コンバージョン**: 各チャネルの特性を活かした誘導
4. **顧客育成**: 段階的な関係構築

## 📊 チャネルミックス戦略

### オウンドメディア（60%）
- **ウェブサイト**: 母艦としての役割
- **ブログ**: SEO・教育コンテンツ
- **メルマガ**: 既存顧客との関係維持

### ソーシャルメディア（30%）
- **Instagram**: ビジュアル訴求・体験談
- **YouTube**: 詳細説明・信頼構築
- **Twitter**: リアルタイム・顧客対応

### ペイドメディア（10%）
- **Google広告**: 検索意図の高い層
- **SNS広告**: ターゲティング精度

## 📱 チャネル別詳細戦略

### 🌐 ウェブサイト

#### 目的・役割
- **ハブ機能**: すべての情報の集約地点
- **コンバージョン**: 最終的な申し込み獲得
- **信頼性**: 公式情報の提供

#### ターゲット
- 検討度の高い見込み客
- 詳しい情報を求める人
- 比較検討中の人

#### コンテンツ戦略
```
トップページ
├── ヘッドライン: 「初めての留学を、最高の思い出に」
├── 3つの特徴（24時間サポート、基礎英語、カスタマイズ）
├── 成功事例（3パターンのペルソナ）
├── サービス詳細へのCTA
└── 無料相談への強いCTA

サービスページ
├── プログラム詳細
├── 料金体系（透明性重視）
├── サポート内容
└── よくある質問

体験談ページ
├── ペルソナ別体験談
├── Before/After
├── 動画インタビュー
└── 関連サービスへのCTA
```

#### KPI
- 月間10,000PV
- 滞在時間3分以上
- コンバージョン率5%
- 直帰率60%以下

### 📝 ブログ

#### 目的・役割
- **SEO**: オーガニック流入の獲得
- **教育**: 初心者の知識向上
- **信頼構築**: 専門性のアピール

#### コンテンツカテゴリー
1. **初心者ガイド**: 基礎知識系
2. **不安解消**: 心理的ハードルを下げる
3. **成功事例**: 共感と希望を与える
4. **実用情報**: 具体的なHow-to

#### 投稿頻度
- 平日: 1記事/日
- 土日: 1記事/2日
- 月間: 25記事目標

#### SEO戦略
```

├── 「初めて 留学」（月間検索数1,000）
├── 「留学 英語できない」（月間検索数800）
└── 「社会人 留学 初心者」（月間検索数500）

ロングテール
├── 「30歳 初海外 留学 体験談」
├── 「英語ゼロ 留学 できる 理由」
└── 「親 反対 留学 説得 方法」
```

### 📸 Instagram

#### 目的・役割
- **ビジュアル訴求**: 留学の魅力を視覚的に
- **コミュニティ**: フォロワーとの交流
- **体験談**: リアルな声の共有

#### ターゲット
- 18-30歳の女性中心
- 視覚的な情報を好む層
- SNSで情報収集する層

#### コンテンツタイプ
```
投稿タイプ別比率
├── 体験談・事例（40%）
├── 教育コンテンツ（30%）
├── サービス紹介（20%）
└── チーム・オフショット（10%）

フォーマット

├── ストーリーズ（リアルタイム）
```

- 土日: 10:00-11:00（朝の時間）

- ハッシュタグ戦略（30個フル活用）
- ユーザー投稿のリポスト
- コメントへの積極的返信
- ライブ配信（月2回）

## 📊 チャネル間連携

### クロスプロモーション
```
ウェブサイト → SNS
├── 各ページにSNSフォローボタン
├── ブログ記事のSNSシェア促進
└── メルマガ登録でSNS限定情報

SNS → ウェブサイト
├── プロフィールリンク最適化
├── 投稿でのブログ記事誘導
└── ストーリーズのスワイプアップ

SNS間連携
├── Instagram → YouTube（動画予告）
├── Twitter → Instagram（投稿紹介）
└── 全SNS → メルマガ（限定情報告知）
```

### 統一されたメッセージング
- ビジュアル: ブランドカラー、フォント統一
- トーン: 全チャネルで同じ話し方
- CTA: 統一されたアクション誘導
