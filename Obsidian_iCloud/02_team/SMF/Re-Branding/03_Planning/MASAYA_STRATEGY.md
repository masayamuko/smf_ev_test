## ⚙️ ワークフロー詳細

### 1. 記事生成プロセス（マサヤ主導）

#### Step 1: コンテンツ企画
```
□ SEOキーワード選定
□ ターゲットペルソナ確認
□ 記事の目的・ゴール設定
□ 人格選択（ヤミー/ユメナ/混合）
```

#### Step 2: AI記事生成
```
□ 人格ファイルをChatGPTに投入
□ 記事アウトライン生成
□ 本文執筆（2,000-3,000文字）
□ 初期SEO最適化
```

#### Step 3: 品質確認
```
□ ブランドガイドライン準拠チェック
□ 事実確認・情報精度検証
□ 読みやすさ・構成確認
□ カウンセラーレビュー依頼
```

### 2. カウンセラーチェック・加工プロセス

#### ヤミー（ブランド戦略記事）
```
□ ブランドメッセージ一貫性確認
□ 専門性・信頼性チェック
□ 競合差別化ポイント強化
□ 最終校正・承認
```

#### ユメナ（顧客体験記事）
```
□ 顧客目線での読みやすさ確認
□ 共感性・親しみやすさチェック
□ 不安解消要素の強化
□ 体験談・具体例の追加
```

### 3. SNS・動画コンテンツ分解（カウンセラー主導）

#### Instagram投稿化
```
□ 記事から核となるメッセージ抽出
□ ビジュアル要素の企画
□ ハッシュタグ選定
□ キャプション作成
```

#### YouTube動画原稿化
```
□ 動画構成（導入・本論・まとめ）
□ 話しやすい口語表現への変換
□ ビジュアル・図解指示
□ CTA（行動喚起）設定
```

#### Twitter投稿シリーズ化
```
□ 記事を5-10ツイートに分解
□ スレッド構成設計
□ エンゲージメント向上要素
□ 投稿タイミング最適化
```

```
【問題】人格抽出が不完全
【対策】段階的ヒアリング・継続改善

【対策】バックアップ体制・代替手段準備

```
