```

初心者専門だから、安心。
まずは無料相談から。
```

**目的**: 不安解消、信頼構築

```
よくある不安、すべて解決します。

✓ 英語力ゼロ → 基礎から教えます
```


## 📝 メッセージ作成のルール

### 必須要素
1. **共感**: 読者の気持ちに寄り添う
2. **解決**: 不安や課題への答え
3. **証拠**: 数字や事例での裏付け
4. **行動**: 次のステップの明示

### 文章構成パターン

#### パターン1: 問題提起型
```
[問題・不安の提示]
[共感の表明]
[解決策の提示]
[行動の促し]
```

#### パターン2: ストーリー型
```
[似た境遇の人の話]
[変化・成功のプロセス]
[読者へのメッセージ]
[行動の促し]
```

#### パターン3: ベネフィット型
```
[得られる未来の提示]
[実現方法の説明]
[サポート体制]
[行動の促し]
```
