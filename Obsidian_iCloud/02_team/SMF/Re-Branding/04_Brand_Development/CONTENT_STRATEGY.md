# コンテンツ戦略

## 🎯 コンテンツ戦略の目的

### 主要目標
1. **認知拡大**: 初心者向け留学の第一想起
2. **信頼構築**: 専門性と実績の証明
3. **不安解消**: 初心者の疑問・不安への回答
4. **行動促進**: 問い合わせ・申込みへの誘導

### KPI設定
- オーガニック流入: 月間1000セッション
- 滞在時間: 平均3分以上
- 問い合わせ率: 5%以上
- コンテンツ経由申込み: 月10件

## 🎨 コンテンツフォーマット

### 記事構成テンプレート
```
1. 導入（共感）- 100字
   読者の悩みや状況に共感

2. 問題提起 - 200字
   なぜこの情報が必要か

3. 本文（解決策）- 1000-1500字
   - 見出し3-5個
   - 箇条書きや表を活用
   - 具体例を豊富に

4. まとめ - 200字
   要点の整理

5. CTA - 100字
   次のアクションへの誘導
```

### ビジュアル要素
- アイキャッチ画像: 明るく前向き
- 図解: 複雑な内容を分かりやすく
- 写真: リアルな留学生活
- インフォグラフィック: データの可視化
