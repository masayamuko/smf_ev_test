# 特定非営利活動法人 青少年の自立を支える福岡の会 - サイトB（ホーム個別重視型）

## 概要

このプロジェクトは、特定非営利活動法人 青少年の自立を支える福岡の会のウェブサイト（ホーム個別重視型アプローチ）です。
各自立援助ホームの個性と特色を最大限活かし、利用者が自分に合ったホームを見つけやすくすることを重視したデザインになっています。

## 特徴

### デザインコンセプト
- **清潔感のあるブルー系**：スチールブルー・スカイブルーを基調とした信頼感のあるカラーパレット
- **各ホームの個性を際立たせる**デザイン
- **ホーム別詳細情報の充実**

### 主要機能

1. **ホーム中心のナビゲーション**
   - 5つのホーム個別紹介
   - ホーム別カラーリング
   - 特色の可視化

2. **ホーム比較・選択支援**
   - カテゴリ別フィルタリング
   - 性別・支援タイプ別表示
   - 選択ガイド

3. **詳細なホーム情報**
   - 各ホーム個別ページ
   - 一日の流れ
   - 支援内容詳細

4. **入居・相談案内**
   - ホーム選択サポート
   - 見学案内
   - 相談フォーム

## ページ構成

```
/                    - トップページ（ホーム紹介メイン）
/about              - 法人について（簡潔な概要）
/homes              - 自立援助ホーム一覧（フィルタリング機能）
/homes/:homeId      - 各ホーム詳細ページ（5つ）
/admission          - 入居・相談案内
/jobs               - 求人情報
/donation           - 寄付・支援
/news               - お知らせ・活動報告
/contact            - お問い合わせ
```

## 特別な機能

### ホームフィルタリング
- **性別別**: 男性向け・女性向け
- **支援タイプ別**: 基本支援・特化支援・次世代型
- **全て表示**: 全ホーム一覧

### ホーム選択支援
- 各ホームの特色を分かりやすく表示
- 比較しやすいカード形式のレイアウト
- 選択ガイドの提供

## 対象ユーザー

1. **利用検討者**：青少年・保護者（ホーム選択重視）
2. **関係者**：児童相談所・学校関係者
3. **支援者**：寄付者・ボランティア・企業

## 開発のポイント

- 各ホームの個性を最大限表現
- 利用者が自分に合ったホームを見つけやすい設計
- 現行の個別サイトからの移行をスムーズに
- ホーム別カラーリングで視覚的差別化

## 他サイトとの違い

### サイトAとの比較
- **A（法人中心型）**: 法人の統一感・信頼性重視
- **B（ホーム個別重視型）**: 各ホームの個性・選択しやすさ重視

### サイトCとの比較  
- **C（ターゲット別アプローチ型）**: 目的別情報設計
- **B（ホーム個別重視型）**: ホーム軸での情報設計
