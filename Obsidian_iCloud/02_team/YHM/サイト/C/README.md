# 特定非営利活動法人 青少年の自立を支える福岡の会 - サイトC（ターゲット別アプローチ型）

## 概要

このプロジェクトは、特定非営利活動法人 青少年の自立を支える福岡の会のウェブサイト（ターゲット別アプローチ型）です。
訪問者の目的（利用検討・支援・求職）に応じて情報を整理し、コンバージョン率を最大化することを重視したデザインになっています。

## 特徴

### デザインコンセプト
- **自然をイメージしたグリーン系**：シーグリーン・ペールグリーンを基調とした成長と希望を表現
- **ユーザージャーニーに基づいた情報設計**
- **目的別ナビゲーション**で使いやすさを重視

### 主要機能

1. **ターゲット別ランディング**
   - 利用をお考えの方へ
   - 支援をお考えの方へ
   - 働きたい方へ

2. **目的特化型コンテンツ**
   - 各ターゲットに最適化された情報提供
   - コンバージョン重視の設計
   - 分かりやすい導線設計

3. **統合されたCTA**
   - 目的別アクションボタン
   - コンバージョン追跡対応
   - A/Bテスト可能な設計

## 技術仕様

- **React** 18.2.0
- **React Router** 6.8.1
- **レスポンシブデザイン**対応
- **コンバージョン最適化**設計

## ページ構成

```
/                    - トップページ（3つの入口を明確に提示）
/for-users          - 利用をお考えの方へ
  ├── 自立援助ホームとは
  ├── 入居までの流れ
  ├── 各ホーム紹介
  └── よくある質問

/for-supporters     - 支援をお考えの方へ（寄付特化）
  ├── 寄付について
  ├── ボランティア募集
  ├── 企業・団体支援
  └── 支援者の声

/for-jobseekers     - 働きたい方へ（求人特化）
  ├── 求人情報
  ├── 職員インタビュー
  ├── 研修・キャリア
  └── 応募方法

/about              - 法人について
/news               - 活動報告・お知らせ
/contact            - お問い合わせ
```

## ユーザージャーニー設計

### 利用検討者
1. **認知** → トップページの「利用をお考えの方へ」
2. **理解** → 自立援助ホームとは何かを学習
3. **検討** → 各ホーム紹介で選択肢を確認
4. **行動** → 入居相談フォームから連絡

### 支援者
1. **認知** → トップページの「支援をお考えの方へ」
2. **共感** → 活動内容と必要性を理解
3. **検討** → 支援方法と効果を確認
4. **行動** → 寄付申込みまたはボランティア参加

### 求職者
1. **認知** → トップページの「働きたい方へ」
2. **興味** → 求人情報と職場環境を確認
3. **検討** → 職員の声と研修制度を確認
4. **行動** → 応募フォームから申込み

## 特別な機能

### コンバージョン最適化
- 目的別ランディングページ
- 明確なCTAボタン配置
- ファネル分析対応設計

### ユーザビリティ向上
- 3ステップでの情報アクセス
- 直感的なナビゲーション
- モバイルファーストデザイン

## 対象ユーザー

1. **利用検討者**：青少年・保護者・関係者（効率的な情報取得重視）
2. **支援者**：寄付者・ボランティア（支援方法明確化重視）
3. **求職者**：福祉関係者・新卒者（応募プロセス簡易化重視）

## 開発のポイント

- ユーザーの目的に沿った情報設計で使いやすさ向上
- 求人・寄付などのコンバージョン率向上が期待
- 情報の整理によりメンテナンス性向上
- 各ターゲットに最適化されたUX提供

## 他サイトとの違い

### サイトA（法人中心型）との比較
- **A**: 法人の統一感・信頼性重視
- **C**: ユーザー目的・コンバージョン重視

### サイトB（ホーム個別重視型）との比較
- **B**: ホーム軸での情報設計
- **C**: 目的軸での情報設計

## 測定可能な指標

- **利用相談**: 問い合わせフォーム送信率
- **寄付**: 寄付申込み完了率
- **求人**: 応募フォーム送信率
- **ページ滞在時間**: 各ターゲットページでの滞在時間
- **離脱率**: ファネル各段階での離脱率
